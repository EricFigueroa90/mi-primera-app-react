# Mi Primera Aplicación en React

## Descripción
Aplicación web desarrollada con React que permite agregar y eliminar tareas.

## Funciones
- Agregar tareas
- Mostrar tareas
- Eliminar tareas
- Contador de tareas

## Tecnologías
- React
- Vite
- JavaScript
- CSS

## Instalación
npm install

## Ejecutar
npm run dev
